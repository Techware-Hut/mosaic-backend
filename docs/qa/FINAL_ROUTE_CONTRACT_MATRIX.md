## Duplicate Mount Note

`routes/vendorOnboarding.routes.js` is mounted at both `/api/vendor-onboarding` and `/admin/vendor-onboard-verify-stage1`. The frontend uses the `/api/vendor-onboarding` surface for the current admin/vendor flows. Both mounts share the same handlers and guards.
